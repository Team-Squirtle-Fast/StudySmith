# StudyView
StudyView is a study tracking application, that helps you refine your skills, track your progress, and keep track of your resources.

-------------Sign Up Page-------------
-Input field to create account 
-Window alerts if an account already exist with this email 
-Window alerts if the username is already taken
-ensures that passwords and confirmed passwords are the same
-Password types are dotted out
